# Paper Assets

This folder contains the latest manuscript assets used for the GeoMERIT paper.

- `latex/`: MDPI BDCC LaTeX manuscript source and references.
- `figures/`: final manuscript figures in PDF/PNG form.
- `scripts/`: plotting scripts for regenerating manuscript figures.
- `data/`: lightweight CSV files used by selected representative-well figures.

The current manuscript title is:

**GeoMERIT: Missingness-Aware and Geological-Cost-Guided Lithology Prediction across Field Wells**
